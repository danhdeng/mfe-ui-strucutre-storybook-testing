# set up Mantine Color Theme

```jsx harmony
<MantineProvider
    withGlobalStyles 
    withNormalizeCSS 
    theme={{colorScheme}}
  >
    <MantineAppShell padding="md">
      <Header
        height={60}
        p="xs"
        sx={{ display: 'flex' }}
        styles={(theme) => ({
          main: {
            backgroundColor:
              theme.colorScheme === 'dark'
                ? theme.colors.dark[8]
                : theme.colors.gray[8],
          },
        })}
      >
        <Title>{title}</title>
      </Header>
      <div>Hello World</div>
    </MantineAppShell>
  </MantineProvider>

```
